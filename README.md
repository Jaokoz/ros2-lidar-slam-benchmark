# ROS 2 LiDAR SLAM Benchmark

Research platform for 2D LiDAR SLAM evaluation in ROS 2 using Gazebo simulation, real robot experiments, ground truth alignment and accuracy benchmarking.

The project includes:
- ROS 2 mobile robot package,
- Gazebo simulation launch files,
- slam_toolbox and Cartographer configuration,
- scripts for running experiments,
- real robot recording pipeline,
- Raspberry Pi server and rover driver,
- ground truth alignment scripts,
- MATLAB analysis script,
- documentation with launch instructions.

Large rosbag files, generated maps, videos and full experimental results are intentionally excluded from this repository.
